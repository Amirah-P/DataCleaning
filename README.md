<h1>Data Cleaning</h1>

<h2>Description</h2>
This Project using this data set to practise. it was then applied to work.



<h2>Languages and Utilities Used</h2>

- <b>Microsoft Excel</b> 
- <b>Microsoft Excel documentation</b>
- <b>Practise data set</b>

<h2>Data Cleaning Steps:</h2>

<b>• Highlighted empty cells with “Conditional Formatting” to identify remaining blanks.</b> 
  
<b>• Removed blank rows using Filter → (Blanks) and deleted them.</b> 

<b>•  Transposed data to adjust layout.</b> 

<b>•  Removed extra spaces using the =TRIM() formula.</b> 

<b>•  Converted text to uppercase using =UPPER().</b> 

<b>•  Cleared unwanted cell formatting.</b> 

<b>•  Removed duplicate entries using “Remove Duplicates”.</b> 

<b>•  Used =LEN() to verify correct character count.</b>
